Npm

The Node package manager (Npm) is a package manager for Node.js packages also called modules that we have used during the semester.

Using Npm we have been able to manage our dependencies by enabling us to install third party modules specified in our package.json file. Resolving dependencies is done very simply through the command line: 

                npm install

Installation of third party modules not specified in the package.json file is also possible. You simply just type the module name after npm install

                npm install express

Npm will make sure to also add this module to your dependeny object inside your package.json file. Npm also has a lot more functionality including the ability to publish your own modules.
