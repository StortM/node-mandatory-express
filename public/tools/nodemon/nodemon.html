Nodemon

Nodemon is a utility tool that we have used during the semester during software development. Nodemon will automatically detect changes in your soruce and automatically restart your server, which makes for a much smoother development process.

After installing Nodemon using the Npm we essentially just have to type a single command to make our software run using Nodemon: 

                nodemon app.js

If software is now running our software will automatically restart every time Nodemon detects changes in the software. This combined witht the VS Code autosave feature makes it so web pages will display your page changes near instantaneously!
